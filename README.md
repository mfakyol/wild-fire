# Wild Fire
A project using the [NASA](https://api.nasa.gov/) api made with React.js.

[Live Demo](http://wild-fire.mfakyol.com)

## Tech
- React.js
- Sass

## Features
- SPA

## Project Setup
```sh
npm install
```

## Project Development Start
```sh
npm run start
```
## Project Build
```sh
npm run build
```
